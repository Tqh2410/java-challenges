# 🚀 14-Day Java Core Challenge

This repository contains all the source code and exercises from my 14-day self-study journey to master the Java Core fundamentals. The goal of this roadmap is to build a strong foundation in syntax, logical thinking, Object-Oriented Programming (OOP), and basic data structures in Java. This will serve as a stepping stone for more complex future projects (such as data processing, building Machine Learning models for house price prediction, etc.).
